# Mood-Tracker-2

Project-1 for JavaScript class, Spring 2026

## Description

Mood-Tracker-2 is a simple web-based application for tracking daily moods and activities. Users can log their current mood using emoji buttons, record up to three activities, and view a history of entries with filtering and statistics.

### Key Features

- **Mood Selection**: Choose from 5 mood options (Happy, Good, Neutral, Sad, Stressed) using intuitive emoji buttons.
- **Activity Logging**: Enter 1-3 activities in a textarea for each entry.
- **Data Persistence**: Entries are saved locally using browser storage.
- **History View**: Display all saved entries with timestamps.
- **Filtering**: Filter entries by mood to focus on specific emotional states.
- **Statistics**: View summary stats like total entries and mood distribution.

## How to Run Locally

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Windows
1. Download or clone the repository to your local machine.
2. Navigate to the `Mood-Tracker-2` folder.
3. Double-click `index.html` to open it in your default web browser.

### macOS
1. Download or clone the repository to your local machine.
2. Navigate to the `Mood-Tracker-2` folder.
3. Double-click `index.html` to open it in your default web browser.

The application runs entirely in the browser with no server required.
